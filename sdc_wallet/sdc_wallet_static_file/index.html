我的钱包
德扑链平台是一个基于区块链技术的短牌德州扑克游戏平台。
采用区块链的技术，解决德州扑克游戏行业的痛点，是对全球线上德州扑克的一项颠覆性创新。
进入我的钱包
平台数字代币SDC(Short Deck Coin)，总发行量 50 亿枚。第一阶段公 开募集 40%，即 20 亿 SDC(25000ETH)。
预售后锁仓 4 个月，首期释放 50%， 剩余每个月解锁 10%。其中:
  • 第一阶段

    20% 基石投资，

    通过后台发放的形式给基石投资人，

    基石轮投资价格 1ETH= 160,000 SDC。

    总募集 5000ETH，最小投资 200ETH。

  • 第二阶段

    30% 天使投资，

    通过网站发放，天使轮投资价格，

    1ETH=115,000 SDC。

    总募集 7500 ETH，最小投资 100ETH。

    等到融资额度到达 7500ETH 后，自动进入下一阶段私募阶段。

  • 第三阶段

    50% 私募投资，

    通过网站发放，私募轮投资价格，

    1ETH= 100,000 SDC。

    募集 12,500 ETH，最小投资 50ETH。

    等融资额度到达 12,500 ETH后，关闭投资管道。
